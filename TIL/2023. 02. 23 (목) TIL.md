### 7. 문자열 보간법과 print 함수
- 문자열을 여러 줄로 표현하고 싶을 때
    
    ```swift
    """
    여기가
    여러줄 문자열이
    들어올 자리입니다
    """
    ```
    
- 문자열 보간법
    
    ```swift
    for count in 1...3 {
    		print("이번엔 \(count) 번째 무야호!!")
    }
    ```
    
- print separator, terminator
    
    ```swift
    print(1, 2, 3, separator: "!")
    //결과 = 1!2!3
    
    print("내가 좋아하는 계절", terminator: " : ")
    print("봄")
    //결과 = 내가 좋아하는 계절 : 봄
    ```
